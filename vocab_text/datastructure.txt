Red Black Tree: max, min, successor, predecessory -
Max: O(lg n)
Min: O(lg n)
Successor: O(lg n)
Predecessor: O(lg n)

Red Black Tree: memory -
Memory: O(n)

Red Black Tree: height -
Height: O(lg n)

Red Black Tree: properties -
1. Every node is either red or black
2. The root is black
3. Every leaf (NIL) is black
4. If a node is red, then both its children are black
5. All simple paths from node to child leaves contain the same # of black nodes

Red Black Tree: advantage, disadvantage -
Advantage: quick insert, delete, and search
Disadvantage: complex implementation

Black height -
# of black nodes, including nil, on the path from given node to a leaf, not inclusive; any node with height h has black-height >= h/2
